# Velocity Drive X

A high-performance web application built with Next.js, TypeScript, and Tailwind CSS.

## Tech Stack
- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS
- Framer Motion (animations)

## Features
- Modern UI design
- Optimized performance
- Scalable project structure
- Reusable components
- Responsive layout
